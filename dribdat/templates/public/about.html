Welcome to

D R I A T _

All your hack belong to us.

This is an open source project board designed for splendid collaboration. On this site we collect and showcase all projects from a hackathon event in one place.

The name is an amalgam of Driven By Data: we started this platform to help hackathon teams sustain their efforts, streamline things for the organisers, and let everyone focus on driving their ideas forward instead of managing IT.

Quickstart

On the front page you can see the upcoming event, as well as any previous events. A short description is followed by a link to the official home page, challenges and any projects submitted, as well as a countdown of time remaining until the start or finish (if already started) of the hackathon.

The Projects & Challenges page is the home of the hackathon. Here you can learn about topics, datasets, and any other information the organisers have provided. Once you have formed a team, you can Start a project after setting up a user account.

When documenting a project, you can use Markdown formatting here, or publish your project using one of the supported open source communities. If you start a project on GitHub, Bitbucket or DokuWiki, you can quickly and easily sync it into Dribdat by entering its link and enabling "auto-update". Then you will find a Sync button at the bottom of your project page for quickly pulling your changes in.

Each project team can pick from one of the announced categories/challenges, or leave the category unspecified. You can state and update anytime the progress level of your project, as well as pick a color and icon to represent yourselves with. Any team members who you would like to list can JOIN the project by clicking the button at the top of your project page. Their public profile will then be linked to it.

We encourage project teams to state their needs and roadblocks using the Community tools built into Dribdat. At the bottom of each project page (if your administrator has set it up), you will be able to make announcements on social platforms. By updating your status, you will also help communicate how far along you are in development, and help hackathon organisers channel their efforts using project scores.

Need more help? Get in touch with the organising team using the Community link above, or visit the School of Data forum.

For organisers

All of the content on the site, including presentation of the header, can be edited by the site administrator in the Admin menu linked in the header if you have access: from where you can edit general details under Events, publish challenges in Categories, and edit or sync Projects.

If you would like to use this platform for your event, contact the organisers of the events listed here and they should be able to help you - or just fork our open source project and set it up to your liking.
